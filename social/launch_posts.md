# Codex教 初回投稿案

## 投稿 1: 発足宣言

Codex教、発足。

OpenAI / ChatGPT / Codex の公式ニュースを、日本語でわかりやすく。
ただ読むだけではなく、Codexで今日どう作れるかまで落とす。

読んで終わりではない。
Codexで作って前へ進める。

## 投稿 2: なぜ作るか

OpenAIの更新は速い。
しかも英語・技術寄り・公式文脈が多い。

だからCodex教では、

- 何が変わったか
- 誰に関係あるか
- 今日どう使うか
- Codexで試すなら何を頼むか

まで日本語にする。

## 投稿 3: 初回ニュース束

2026-05-27時点で、Codexまわりの重要公式ニュースを初回整理。

- Codex app
- Codex for almost everything
- Codex安全運用
- Windowsサンドボックス
- Appshots / Goal mode / リモート継続

ここから日本語で噛み砕いていく。

## 投稿 4: 非公式宣言

Codex教はOpenAI公式ではありません。

公式ソースを読み、日付とURLを残し、日本語で実用解説する非公式メディアです。
噂は噂、推測は推測、公式は公式として扱う。

正道で勝つ。

## 投稿 5: 会員導線の種
